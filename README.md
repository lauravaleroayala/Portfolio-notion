# Portfolio

Este repositorio contiene mi portfolio profesional exportado desde Notion.

## Cómo está organizado

### Home (`index`)

La página principal incluye:

- Un breve apartado **About Me** con mi perfil profesional.
- Seis tarjetas (cards), cada una enlazada a un proyecto diferente.
- Cuatro llamadas a la acción (CTA) al final de la página.

## Proyectos

Cada proyecto es un caso de estudio independiente.

Todos siguen la misma estructura:

1. Contexto del proyecto.
2. Problema o necesidad detectada.
3. Proceso de investigación y análisis.
4. Solución propuesta e implementada.
5. Resultado e impacto obtenido.
6. Aprendizajes relevantes (cuando aplica).

El objetivo de cada proyecto es explicar el razonamiento seguido para resolver un problema real, más que mostrar únicamente el resultado final.

## About Me

La sección "About Me" resume:

- Perfil profesional.
- Especialización.
- Experiencia.
- Forma de trabajar.
- Intereses profesionales.

## Enlaces (CTA)

Al final de la página principal aparecen cuatro enlaces:

## Cómo interpretar este portfolio

Cada proyecto debe entenderse como un caso práctico de resolución de problemas.

Al analizar un proyecto es importante priorizar:

- El problema inicial.
- El proceso seguido para entenderlo.
- Las decisiones tomadas.
- La solución implementada.
- El impacto conseguido.

El portfolio está pensado para mostrar metodología, capacidad de análisis y proceso de diseño, no únicamente el resultado visual.
